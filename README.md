# PyJarvis AI Assistant

PyJarvis is a Python-based AI assistant inspired by Jarvis,
with a React frontend for voice/chat interaction.

## Features
- Voice & text interaction
- Web search
- Weather updates
- File & window control
- Keyboard & mouse automation
- Memory storage & recall
- React-based UI

## Tech Stack
- Python
- OpenAI / AI logic
- React
- Node.js

## Setup
1. Clone the repo
2. Install Python requirements
   pip install -r requirements.txt
3. Start backend
   python agent.py
4. Start frontend
   npm install
   npm start
